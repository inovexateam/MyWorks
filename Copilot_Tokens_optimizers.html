Invisible token leaks. Inflate bill before you type one word.

📑
Open editor tabs = hidden context
Every open file silently added as context. 20 tabs = up to 100K tokens before you ask anything.
Fix: close unrelated tabs. Keep only the file you're editing.
💬
Long chats re-send history every turn
Each message resends full conversation as input. 20-turn chat = 5× more tokens for same final question.
Fix: new chat per task. Sessions = sticky notes, not novels.
🔧
copilot-instructions.md loads every agent step
LLM boilerplate adds 2K–5K tokens/step. 15 steps × 5K = 75K overhead tokens.
Fix: trim to <300 tokens. Constraints only. Delete anything obvious.
🔌
MCP servers: 100–500 tokens each, every step
15 servers × 15 steps = 112,500 tokens of pure overhead before task runs.
Fix: 3–5 active MCP max. Remove rest.
📦
node_modules / generated files get indexed
Thousands of files, zero useful signal. Expand repo context for nothing.
Fix: add node_modules/ dist/ .cache/ logs/ to .copilotignore
🖱️
No selection = full repo index search
Copilot Chat without selection may search entire codebase. Highlighting 10–20 lines scopes precisely.
Fix: select relevant lines before opening Chat. 80%+ input token reduction.
🏗️
Multi-service repos open simultaneously
Files from multiple microservices open at once multiply context across unrelated code.
Fix: VS Code workspaces or separate windows per service.
